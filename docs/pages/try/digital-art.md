# デジタルアートをやってみよう！
皆さんが、スマートフォンやパソコンでインターネットにあるページを見たりするときに使う「ブラウザ」。  
このブラウザの上で動く、シンプルなデジタルアートを作るやり方を、紹介していければと思います。  

この後の説明を見ながら進めていくと、例えば、このようなデジタルアートを作ることができます。

【★画像か動画：サンプル】

まずは、図形を描くやり方を試すことから始めて、その後にはパソコンのマウス・タッチパッドの操作に反応して動きを変えるアートを作ってみましょう！

## さっそく試してみる
いろいろな説明をする前に、まずは図形を描く簡単なやり方を、実際に試してみようと思います。

### プログラムを書くところについて
この後、デジタルアートを描くのに「[p5.js](https://p5js.jp/)」というものを使います。  
これは、JavaScriptというプログラミング言語で使えるものです。

今は「そういう名前のものを使うんだな」という感じで、名前だけ覚えてもらっていれば大丈夫です。

ここから、このp5.jsを使ったプログラムを作っていくのですが、そのやり方がいくつかあります。

例えば、自分のパソコンの中でプログラム用のファイルを作って、書いていくやり方があります。  
その他にも、最初に名前を出していた「ブラウザ」を使って書いていくやり方もあります。  

ここでは、2つ目に書いたやりかたを使っていきます。  
具体的には、ブラウザの上でp5.jsのプログラムを書いたり、動かした結果を見ることができる「[p5.js Web Editor](https://editor.p5js.org/)」というものを使います。

### プログラムの説明： 丸を描いてみる
それでは、先ほど試しに書いたプログラムについて説明をしていきます。

#### 1回だけの処理と繰り返される処理
先ほど書いたプログラムを見てみると、どちらも「setup()」と「draw()」という2つの部分があるのが分かります。

「setup()」は最初に1回だけ行われる処理で、「draw()」のほうは何度も繰り返し行われる処理になります。  
この話について、最初に書いたプログラムでもう少し説明します。

#### 1回だけの処理
最初に書いたプログラムの「setup()」では、「createCanvas(400, 400)」という行と、「background(220)」という行があります。

```javascript
function setup() {
  createCanvas(400, 400);
  background(220);
}
```

「createCanvas(400, 400)」という部分は、キャンバスと呼ばれるデジタルアートを書く場所を作っています。  
400という数字が2つありますが、これはそれぞれ「キャンバスの縦の長さ」と「キャンバスの横の長さ」を示しています。

試しに、この数字を減らして200などにしてみたり、逆に増やしてみたりすると、どんな意味なのかが分かりやすいのではないかと思います。

そして、「background(220)」という部分は、先ほど作ったキャンバスの背景の色を決めるものです。  
220という数字が色を決めるためのものになっています。

ここの数字を0や100、255といった数字に変えてみて、背景の色がどのようになるか確かめてみてください。  
また、「background(0, 0, 200)」や「background(200, 0, 0)」、「background(0, 200, 0)」などと3つの数字を書いてみましょう。

色を指定する数字が一つの時は、白黒や灰色になり、3つの数字を指定した時は赤や青など色がついたものになったのではないでしょうか。  
数字を一つだけにした時は、数字の大きさを変えていくと真っ黒（数字でいうと0）から真っ白（数字でいうと255）、その間の灰色となります。  

そして、3つの数字を書いた場合は、1つ目から3つ目がそれぞれ「RGB（赤・緑・青）」を指定することになります。  
この時、「background(200, 0, 200)」や「background(0, 200, 200)」、「background(100, 150, 100)」などとすると、赤・緑・青の色を混ぜたものを使うことができます。

#### 繰り返される処理
今度は「draw()」の中を見ていきます。

```javascript
function draw() {
  fill(50, 100, 200);
  ellipse(200, 200, 50, 50);
}
```

「fill(50, 100, 200)」という部分は、キャンバスの真ん中あたりに描いた円の中の色（塗りつぶしの色）を指定しています。  
先ほど、「background()」の数字を変えてみた時と同じように、3つの数字の部分を自分の好きな数字に変えてえみてください。

その後に出てくる「ellipse(200, 200, 50, 50)」はだ円という丸を描くためのものです。  
だ円というのは、縦と横の長さが違う丸になります。

「描かれた丸は縦と横の長さが同じだったのでは？」と思われたかもしれません。  
これについて、説明をしていきます。

そのために「ellipse(200, 200, 50, 50)」で書いている4つの数字の意味を、最初に少し補足しておきます。  
これは順番に「丸の中心のx座標」、「丸の中心のy座標」、「丸の横方向の長さ」、「丸の縦方向の長さ」を指定するものです。

これも「ellipse(100, 200, 50, 50)」や、「ellipse(200, 200, 100, 50)」などといったように、数字を変えてえみると理解しやすいと思います。

円の中心の位置についてですが、最初に「createCanvas(400, 400)」という部分で縦も横も400の大きさのキャンバスを作っているため、「200, 200」という数字を指定すると、これはキャンバスの中心の位置を指定することになります。  
また、だ円の縦と横の長さについて「50, 50」という指定をしていたため、縦も横も同じ大きさの丸になっていたというわけです。  
ちなみに、「ellipse(200, 200, 50, 50)」は「ellipse(200, 200, 50)」というように、4つ目の数字を省略しても同じ意味になったりもします（省略した4つ目の数字の部分に、3つ目の数字が使われることになります）。

### プログラムの説明： マウスの動いたところに丸を描く
今度は、2つ目に試したプログラムについて、説明をしていきます。  
なお、1つ目のプログラムで説明した部分は、ここでは省略します。

#### 1回だけの処理

```javascript
function setup() {
  createCanvas(400, 400);
  background(220);
  noStroke();
}
```

ここで登場している「noStroke()」は、図形の線を描かない、という意味になります。

2つ目のプログラムも、1つ目のプログラムと同じように「draw()」の中で「ellipse()」を使っています。  
最初の部分で「noStroke()」を書いていると、1つ目のプログラムの時と違って、丸を描いた時に外側を囲んでいた線がなくなり、塗りつぶしの色だけが描かれることになります。

#### 繰り返される処理

```javascript
function draw() {
  fill(random(0,50),random(50,150),random(100,200), 50);
  ellipse(mouseX, mouseY, 50, 50);
}
```

ここで使っている「fill()」の中で「random()」という部分があります。  
これは「ランダムな数字（乱数）」を使うもので、デジタルアートではよく使われます。

この中の2つの数字を書いている部分は、ランダムに選ばれる数字の最小値と最大値をしていするものです。  
例えば「random(0,50)」は、0から50までの数字が何か選ばれます。  
この時、0は選ばれる数字に含まれますが、50は選ばれる数字に含まれません。

また、ここで出てくるランダム数字は、整数ではなく小数になります。

それと、2つ目のプログラムの「fill()」では、1つ目のプログラムでの「fill()」と違って、4つ目の数字を書いています。  
「random(0,50),random(50,150),random(100,200)」という3つの数字の後に「50」という数字が書かれているのが分かります。

これは、透明度を指定するもので、1つ目のプログラムのように何も指定していないと「100を指定したのと同じ不透明」となります。  
そして、この部分で書く数字が0に近いほど、後ろが透ける透明な色になっていきます（0にすると、完全に透明になるため、色が何も見えない状態になります）。

あとは、最後の行の「ellipse()」の説明です。  
1つ目のプログラムと違って、最初の2つの数字を指定する部分が「mouseX, mouseY」となっています。  
これは、キャンバス上にマウスカーソルがあった場合に、そのマウスカーソルのx座標とy座標を示すものになります。  
そのため、キャンバスの上でマウスカーソルを動かすと、丸が描かれる位置が、マウスカーソルに合った位置になっていたわけです。

この「mouseX, mouseY」は、2つ目のプログラムのように、簡単にマウスカーソルの移動に連動した描画を使うことができ、とても便利です。

## まとめ
今回、p5.jsを使ってデジタルアートを描くというのを試しました。

このp5.jsに関して、今回使った図形を描く・マウスカーソルの位置に合わせて描画をする等といった機能は、そのごく一部のものです。  
その他にもデジタルアートを描くための便利な機能があったり、2Dの平面上のアートだけでなく、3Dの立体的な図形・アートを描くこともできます。  
また、カメラの映像を使う、といったこともできる仕組みがあります。

もし、「もっといろいろなデジタルアートを描いてみたい！」という場合は、「参考になるページ」の部分に掲載したページを見てみたり、自分で他の情報を調べたりして、いろいろな自分の好きなデジタルアートを作ってみてください！

なお、p5.jsはJavaScriptで使えるものですので、p5.js以外のJavaScript用の仕組みと組み合わせることもできます。  
どのようなものがあるか、動いている様子の紹介という形になりますが、最後にいくつか例を掲載してみようと思います。
